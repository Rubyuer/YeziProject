# 两片叶子

  "两片叶子"是一款线上购买盆摘的应用。
  
  【下单流程】 
  1.选择心仪商品-商品详情-立即下单,进行单件商品的购买方式;
  2.选择心仪商品-商品详情-加入购物车,在购物车进行批量下单。
  
  【获取方式】
  商品快递到家,支付获取商品。
  
  【盆摘种类】 
  观叶植物、观花植物、多肉/仙人掌、景观/奇趣类盆摘。

  【科学种植】 
  科学养殖绿色植物,精心选种,科学选取容器,只为提供更好看的绿色植物。
  
  
  身边多一点绿色,心情多一点彩色。
